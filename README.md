## 用語集

### GLSL

GLSLはOpenGL Shading Language、GLslangとしても知られる。
C言語構文ベースとした高レベルシェーディング言語である。

アセンブリ言語やハードウェアに依存した言語を使わず、グラフィックスパイプラインを直接制御できるように策定された。
